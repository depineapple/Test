Входными данными для программы являются файлы replacement.json, находящийся в корне программы по пути ...\Test\ConsoleApp1\bin\Debug, а также
файл data.json, получаемый при помощи HTTP GET запроса.
Для получения файла result.json необходимо запустить консольное приложение и дождаться его выполнения. В результате будет сформирован необходимый
файл, содержащий исправленные сообщения, представленные в виде массива строк. Полученный файл также находится в корневом каталоге 
по пути ...\Test\ConsoleApp1\bin\Debug.
